.. _page-tsconfig:

Page TSconfig
-------------

Upon installation, the extension will set default properties in Page
TSconfig as specified by the extension configuration variable:
**Default configuration settings**. Three default configurations are
available: Minimal, Typical, and Demo. These default configurations are
documented in the next section of the present document.

These properties may be modified in the Page TSconfig of your page;
this is most often done on the root page of your site.
