.. _manual_import:

=============
Manual import
=============

.. note::

   Make sure that any required extensions are installed and the database scheme
   is up-to-date before starting the import. Otherwise the data related
   to non-existing tables will not be imported.

.. rst-class:: bignums

   1. Upload the export file

      Upload the file to your destination installation. Just like the
      export module you can find the import module in the page tree context menu
      :guilabel:`More options... -> Import`.

      .. include:: /Images/AutomaticScreenshots/UploadImport.rst.txt

   2. Preview the imported data

      A tree with the records to be imported is displayed
      automatically. If you change any of the options you can reload
      this display with the :guilabel:`preview` button.

      .. include:: /Images/ManualScreenshots/ImportPreview.rst.txt

   3. Import

      Click the import button.
