=============================
PSR-14 events in EXT:lowlevel
=============================

The following PSR-14 event is available to extend the functionality:

..  _psr14events-ModifyBlindedConfigurationOptionsEvent:

ModifyBlindedConfigurationOptionsEvent
======================================

The event allows to blind (hide) any configuration options. Usually such options
are passwords or other sensitive information.
:ref:`More details <t3coreapi:ModifyBlindedConfigurationOptionsEvent>`
