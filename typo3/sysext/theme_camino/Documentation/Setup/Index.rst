=======================================
Setting up a web site with Camino theme
=======================================

After Camino is installed and activated it needs to be loaded.

..  contents::

..  _page-tree:

Page tree
=========

Ensure that at least one page exists in the Page Tree and the root / parent page
is configured correctly: :guilabel:`Root Page > Page Properties > Behaviour > Use as Root Page`

..  figure:: /Images/SetupRootPage.png
    :class: with-border
    :alt: Screenshot of the Page Properties behaviour tab
    :zoom: lightbox

When a Root Page is created, a new Site record is automatically created.

..  _edit-site-configuration:

Edit Site Configuration
=======================

Access the `Setup` module: :guilabel:`Sites > Setup > Edit Site Configuration (pencil icon) > General`

..  figure:: /Images/SetupPencilIcon.png
    :class: with-border
    :alt: Screenshot of the Site > Setup module
    :zoom: lightbox

..  _load-camino:

Load Camino
===========

Select the Camino theme and add it to the `Selected Items` column.

..  figure:: /Images/SetupLoadCamino.png
    :class: with-border
    :alt: Screenshot of the Site Configuration > General tab
    :zoom: lightbox
