Configure the image processing tool used by TYPO3.

TYPO3 uses GraphicsMagick or ImageMagick for advanced image processing. The correct configuration depends on the version available on your system. Available image processing tools are detected automatically and appropriate settings are recommended.
When configured correctly, image processing tests in the Install Tool → Test Setup section should pass.

If the ImageMagick or GraphicsMagick executables are installed in a non-standard location, specify the absolute path to the directory here.
