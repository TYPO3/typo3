Configure the password hashing algorithm used for frontend and backend users.

TYPO3 never stores passwords in plain text. Instead, passwords are stored using a salted one-way hash. Available hashing algorithms are detected automatically.

If unsure, select the first available option, which represents the most secure algorithm supported by the system.
The selected algorithm is used for new backend and frontend users. Existing users are automatically upgraded to the new algorithm after their next successful login.
