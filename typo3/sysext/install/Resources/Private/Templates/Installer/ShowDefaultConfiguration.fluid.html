The Installation Wizard can create a starter website with the default TYPO3 theme. This includes a root page, site configuration, and frontend rendering ready to use.

The Installation Wizard can create a starter website with basic frontend rendering. This includes a root page, site configuration, and content element support.

Pre-configured backend user groups

If you would like to have pre-configured and ready to use base backend user groups "Editor" and "Advanced Editor", select this option. You can rename these preset backend user groups after their creation.
