The Install Tool is locked
  • If you are installing TYPO3, please create the file FIRST_INSTALL in the document root of your web server. New to TYPO3? We recommend reading the Getting Started Documentation.
  • For an existing TYPO3 installation, please create the file ENABLE_INSTALL_TOOL in the directory {enableInstallToolPath}.

Either file must be writable by the web server user. The filename is case-sensitive. The file can be empty.

Security note: When you are finished with your task, you should rename or delete any FIRST_INSTALL and ENABLE_INSTALL_TOOL files. They will automatically be deleted if you log out of the Install Tool or if the file is older than one hour.
