====================================================================
Feature: #56529 - Support \"has*\" Functions in extbase ObjectAccess
====================================================================

See :issue:`56529`

Description
===========

ObjectAccess supports \"is*\" and \"get*\" functions to return
property values. Another convenience method is \"has*\", which
can be used in the same way.
